# CS2_DoubleJump
CS2 DoubleJump - плагин для Counter-Strike 2, позволяющий игрокам выполнять дополнительные прыжки в воздухе

## Описание плагина: CS2 DoubleJump - плагин для CS2, добавляющий возможность дополнительных прыжков в воздухе (двойной прыжок и воздушный отскок при падении).

### Конфигурационные параметры:
css_doublejump_enabled (true) - Глобальное включение плагина

css_doublejump_boost_units (300.0) - Базовая высота прыжка

css_doublejump_max_jumps (1) - Максимум доп. прыжков (1-5)

css_doublejump_air_jump_enabled (true) - Прыжки в воздухе при падении

css_doublejump_min_velocity_for_air_jump (-50.0) - Мин. скорость для воздушного прыжка

css_doublejump_air_jump_boost_multiplier (1.0) - Множитель силы воздушного прыжка


### Консольные команды:
css_doublejump_help - Показать справку

css_doublejump_settings - Показать настройки

css_plugins reload CS2DoubleJump - Перезагрузить конфигурацию
